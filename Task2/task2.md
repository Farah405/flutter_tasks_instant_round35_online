# Task 2: Find Maximum of Three Numbers

Write a Dart program that reads three integers (int a, b, c) from the user, then prints the largest number among them.

## Requirements:
- Read three integer inputs from the user
- Find and display the largest of the three numbers
- Handle the input appropriately

## Example Usage:
```
Enter first number: 10
Enter second number: 25
Enter third number: 15
The largest number is: 25
```

## Implementation Notes:
- Use Dart's standard input methods to read user input
- You can use comparison operators and conditional statements
